# 👁 VisionAid AI

### See The World. Through Sound.

VisionAid AI is an AI-powered accessibility platform designed to help blind and visually impaired users understand their surroundings independently.

The platform uses computer vision, voice technology, and AI assistance to identify objects, describe environments, and communicate information through natural voice output.

# ✨ Features

## 📷 AI Object Scanner

* Camera-based object scanning
* Object identification
* Living & non-living classification
* Smart descriptions
* Real-time visual assistance

Examples:

Living:

* Humans
* Animals
* Birds
* Plants

Non-Living:

* Electronics
* Books
* Vehicles
* Food
* Documents
* Daily objects

## 🔊 Voice Assistant

VisionAid AI provides audio feedback using:

* Text-to-Speech
* Voice commands
* Hands-free interaction
* Spoken object descriptions

Users can ask:

* "What is this?"
* "Describe my surroundings"
* "Read this text"
